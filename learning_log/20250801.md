# For - Loop 예시
1. 구조
2. 
```java
package ch07_loops;

public class Loop06 {
    public static void main(String[] args) {
        for( int i = 0 ; i < 6 ; i++ ) {
            for( int j = 0 ; j < i ; j++) {
                System.out.print("*");
            }
            System.out.println();
        }

        for (int i = 0 ; i < 5 ; i++) {
            for (int j = 5 ; j - i > 0 ; j--) {
                System.out.print("⭐");
            }
            System.out.println();
        }
        
        for (int i = 5 ; i > 0 ; i--) {
            for(int j = 0 ; j < i ; j++) {
                System.out.print("❤️");
            }
            System.out.println();
        }
        
        for(int i = 100 ; i > 0 ; i--) {
            System.out.println(i);
        }

    }
}
```
